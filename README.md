# ChessPersonalTrainer

Personalized chess move recommendations that blend engine evaluation with design-centric connectivity heuristics.

## Project Goal
Build a Personalized Logic Filter that ranks moves by user-specific cognitive strengths
(Connectivity, Response, Influence) in addition to raw engine evaluation.

## Core Ideas
- **Universe A (Engine):** Stockfish top-line moves
- **Universe B (Design):** Moves that maximize connectivity between fragmented piece clusters
- **Delta-V:** Acceptable evaluation loss to gain structural clarity

## Status
Scaffolding in progress.
